# Описание
Приложение для поиска и отображения информации о кинофильмах, с поддержкой Android и iOS.<br>
Приложение позволяет искать фильмы по названию или по имени одного из актеров, а также устанавливать фильтр с необходимыми жанрами.<br>
### Главный экран
На главном экране находится строка поиска и фильтр.<br>
<img width="471" height="962" src="https://github.com/user-attachments/assets/6fa2e038-76d4-4379-bdfb-ed0b3fd9ced0" />
В развернутом фильтре можно выбрать режим поиска: по названию или по имени одного из актеров. Также в списке фанров можно выбрать те, которые обязательно должны присутствовать в фильме.<br>
Ниже строки поиска и фильтров находится список найденных фильмов.
### Экран с деталями о фильме
На экране выбранного фильма находится детальная информация о нем. Такая как: название, дата премьеры, жанры, актеры и описание фильма. В верху экрана находится кнопка "Назад".<br>

# Порядок развертывания
### Загрузка проекта
- Установите Visual Studio 2022 версии 17.X или новее
- Для нужной версии Visual Studio установите рабочую нагрузку "Разработка с помощью .NET Multi-Platform App UI" с помощью Visual Studio Installer.
- Создайте пустой каталог.
- Откройте терминал и перейдите в созданный каталог.
- Выполните команды:
  ```
  git clone https://github.com/Vefery/MauiFilmApp.git
  cd cd MauiFilmApp/FilmsApp
  start FilmsApp.sln
  ```
- Дождитесь загрузки всех необходимых зависимостей.

### Сборка для Android
- Выбирите конфигурацию "Release" и целевую платформу "net9.0-android"<br>
  <img width="632" height="327" src="https://github.com/user-attachments/assets/bab55982-81fd-4fb8-9a50-e221ca6ede41" />
- Найдите "FilmsApp" в обозревателе решений, нажмите правой кнопкой мыши и выбирите "Опубликовать...". Дождитесь завершения архивации пакета.
- Нажмите <b>Распостранение... > Произвольный</b>, создайте удостоверение подписания и сохраните .apk файл в любую удобную директорию.

### Сборка для IOS
- Установите XCode.
- Выбирите конфигурацию "Release", целевую платформу "net9.0-ios" и выбранное устройство <b>Удаленные устройства IOS > Удаленное устройство</b>.<br>
  <img width="627" height="328" src="https://github.com/user-attachments/assets/6bd4fcf2-8ccf-4a0f-9d26-86491ebf1948" />
- Создайте сертификат распостранения в <b>Средства > Параметры > Xamarin > Учетные записи Apple > Имя вашей учетной записи</b>.
- Добавьте устройства, на которых будет запускаться приложение, к вашей учетной записи в предыдущего шага используя XCode.
- Найдите "FilmsApp" в обозревателе решений, нажмите правой кнопкой мыши и выбирите "Опубликовать...". Дождитесь завершения архивации пакета.
- Нажмите <b>Распостранение... > Произвольный</b>, выбирите <b>Профиль подготовки</b> и <b>Идентификатор подписи</b>, затем сохраните .ipa файл в любую удобную директорию.
